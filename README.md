# Yukang Chen — Personal Website

个人主页、项目展示与博客，采用 Bay Jekyll Theme 的简约风格，并通过 GitHub Pages 发布。

## 更新内容

- 编辑 `_config.yml` 修改网站名称、导航和社交链接。
- 编辑 `index.md` 修改首页介绍。
- 编辑 `projects.md` 添加或更新项目。
- 在 `_posts/` 中新建 `YYYY-MM-DD-title.md` 格式的文件来发布文章。
- 主题源样式位于 `_sass/`，网站实际加载 `assets/css/main.css`。

推送到 `main` 分支后，GitHub Pages 会自动构建并发布网站。

主题基于 [Bay](https://github.com/eliottvincent/bay)，许可信息见 `BAY_THEME_LICENSE.md`。
